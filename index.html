Политика конфиденциальности

Обновлено: июнь 2025 года

Настоящая Политика конфиденциальности регулирует порядок сбора, хранения, обработки и использования персональных данных пользователей мобильного приложения, предназначенного для сотрудников страховой компании «ATN Polis».

1. Цель приложения

Приложение используется аварийными комиссарами страховой компании для оформления первичной информации о страховых случаях на месте происшествия.

2. Сбор и использование данных

Приложение может собирать следующие данные:

  • Фотографии, сделанные пользователем в ходе осмотра;
  • Геолокация (если включена) для фиксации адреса происшествия;
  • Данные полиса, связанного с происшествием;
  • Информация об авторизованном пользователе (ФИО, служебный ID).

Все данные обрабатываются исключительно в служебных целях и передаются в корпоративную систему страховой компании.

3. Доступ и безопасность

Доступ к приложению осуществляется через авторизацию. Регистрация пользователей происходит централизованно через административную панель компании. Все данные защищены средствами аутентификации и передаются по зашифрованным каналам связи.

4. Раскрытие информации

Личные данные не передаются третьим лицам, за исключением случаев, предусмотренных законодательством Кыргызской Республики.

5. Права пользователя

Пользователь вправе запросить:

  • Информацию о своих данных, хранящихся в системе;
  • Удаление или исправление данных (через администратора компании).
